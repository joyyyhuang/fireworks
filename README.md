# 烟花动画 🎆

一个精美的交互式烟花动画网页，点击或拖动屏幕即可释放烟花！

## 功能特点

- ✨ 点击或拖动屏幕释放烟花
- 🎨 随机生成的多彩烟花效果
- 🎵 支持背景音乐（首次交互后播放）
- 📱 响应式设计，支持移动端
- 🌙 精美的夜空渐变背景
- 💫 自动播放烟花效果

## 使用方法

1. 在浏览器中打开 `fireworks.html`
2. 点击或拖动屏幕释放烟花
3. 首次点击后会播放背景音乐

## 技术栈

- HTML5 Canvas
- 原生 JavaScript
- CSS3
